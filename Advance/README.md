# Advanced Python Programs

This folder contains some advanced Python programs that I created while learning Python.
These projects use different Python libraries and concepts such as networking, automation, APIs, and file handling.

The purpose of these programs is to practice building real-world tools instead of simple beginner scripts.

## Programs in this folder

* **web_scraper.py** – Extracts information from websites.
* **port_scanner.py** – Checks open ports on a target system.
* **password_manager.py** – Stores passwords in a file.
* **weather_app.py** – Shows weather information for a city.
* **file_organizer.py** – Automatically sorts files by type.
* **system_monitor.py** – Displays CPU and memory usage.
* **url_status_checker.py** – Checks if websites are online.
* **image_downloader.py** – Downloads images from a URL.

## Requirements

Some programs require external libraries.

Install them using:

pip install requests beautifulsoup4 psutil markdown

## Running the programs

Run any script using Python:

python filename.py

Example:

python web_scraper.py
